# NextJS Introduction

##### prettier 각 프로젝트마다 적용 방법

1. setting.json에 하위 내용 추가

```
{
    "[javascript]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
    },
    "editor.formatOnSave": true,
}
```

2. 프로젝트 폴더에 .preetierrc 파일 추가 후 prettier 내용 기입
